# resize_window-一个开箱即用的能够帮助你控制窗口大小的软件
一个开箱即用的能够帮助你控制窗口大小的软件（A software that helps you control the window size out of the box）

你是否每次办公、游戏、分屏都因为窗口难以调整大小而发疯？也许这个开箱即用的python开源小软件能帮到你！
Are you driven crazy by the difficulty of resizing windows every time you're working, gaming, or using split-screen? Perhaps this small, out-of-the-box Python open-source software can help!

你可以直接下载dist里面的.exe文件，直接运行就能够调整窗口大小！如果你希望将其修改成更符合你胃口的软件，可以直接下载.py文件，其中一个是应用文件，另一个是打包文件！
You can directly download the .exe file in dist and run it directly to adjust the window size! If you want to modify it into a software that suits your taste, you can directly download the .py file, one of which is the application file and the other is the package file!
